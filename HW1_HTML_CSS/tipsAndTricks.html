Common Mistakes

  1. Forgetting to Check Compatability: Not all parts/components work together. The CPU must match the motherbaord socket and the RAM tyoe must also be supported by the motherboard.
  2. RAM Not Fully Seated: RAM sticks need to bu fully seated and clicked into place otherwise the PC may not boot.
  3. Forgetting the CPU Power Cable: You may have plugged in the motherboard to get power but the CUP also needs its own power that is near the top of the board.
  4. Plugging Monitor Into the Wrong Port: When you use a dedicated graphics card, the monitor must be connected to the CPU and not the motherboard itself.
  5. Installing the Motherboard Without Standoffs: Standoffs help prevent the motherboard from touching your metal case, if the motherboard is not on a standoff then it can short circuit, ruining the motherboard.
  6. Poor Airflow: Fans installed poorly or in the wrong direction can trap heat inside, which can cause overheating.

Quick Tips

  • Read the motherboard manual before connecting cables.
  • Install the CPU, RAM, and storage on the motherboard before placing it in the case.
  • Keep screws organized to avoid losing them.
  • Do not overtighten screws; snug is enough.
  • Use cable management to improve airflow.
  • Install system drivers after setting up the operating system.
