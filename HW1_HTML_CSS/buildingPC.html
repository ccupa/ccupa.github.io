Steps to Building your PC

Installing your parts in a safe order and double checing connections is the most important part of building a PC. Depending on the case you chose steps can very but are more or less the same process.

Assembly Steps

  1. READ ALL INFORMATION BEFORE ATTEMPTING ON YOUR OWN
  2. Prepare your workspace: Work on a clean surface and avoid building on anything that can create static electricity
  3. Install CPU onto Motherboard: Carefully line up the CPU with the socket and lock into place. For more depth on how to do so watch this video. How to Install CPU onto Motherboard
  4. Install Ram onto Motherboard: Open RAM slot clips press sticks down until the click into place. Be careful since these can break if too much pressure is applied. Video: How to Install RAM onto Motherboard
  5. Install SSD into M.2 slot (if you have one): Slide SSD into the slot on motherboard and secure with case screw.
  6. Mount Motherboard into case: Place onto case standoffs and screw it in (DO NOT OVER TIGHTEN).
  7. Install the Power Suppply: Secure into case and route the cables where they need to go.
  8. Install storage drives(SATA SSD/HDD): Mount drive in the case drive bay or bracket, then connect the data cable to the motherboard and power to PSU cable.
  9. Install Graphics Card (GPU): Insert GPU into the PCIe slot on the motherboard adn secure it to your case.
  10. Connect all Power and Front Panel Cables: Plug in all components, 24-pin motherboard power, CPU power, GPU power, and the case front-panel connectors (power button, USB, audio).
  11. Check cables and Airflow: Make sure fans can spin freely and are not blocked.
  12. First Boot Test: Turn on PC and enter BIOS/UEFI to confirm the components are detected.
  13. Instal OS and Drivers: Install windows/linux, then install motherbaord and GPU drivers.

Common First-Boot Checks

  • Make sure monitor cable is connected to GPU and not the motherboard (a very common mistake).
  • Confirm RAM is fully clicked into place.
  • Check that power cabled and fully seated and that power to the PC is turned on.
